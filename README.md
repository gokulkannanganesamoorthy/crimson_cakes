A premium, high-performance portfolio website for Crimson Cakes, an artisan bakery brand. This project showcases the brand's exquisite cake designs through a modern, "cool" aesthetic, featuring advanced web technologies for a seamless user experience.

## ‚ú® Features

- **Cinematic Hero Section**: Immersive parallax storytelling with 3D-style floating visuals.
- **Cool Bento Gallery**: A visually stunning, interactive gallery showcasing the cake collection with "spotlight" focus effects.
- **Smooth Scrolling**: Optimized scrolling experience with parallax layers (Process, FAQ, Footer) that slide over each other.
- **Responsive Design**: Fully responsive layout that looks stunning on all devices, from mobile to ultra-wide screens.
- **Premium Aesthetics**: Curated color palette (Crimson & Stone), custom typography, and glassmorphism effects.

## üöÄ Getting Started

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/iamgokulkannan/crimson-cakes.git
    cd crimson-cakes
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Run the development server:**

    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
